# MemoryVault — Folder Guide

Reference map of this memory store. Not instructions for Claude Code — see root CLAUDE.md for that.

## Folders

`medias/` — consumed media, organized by type
  - `ecoutes/` — podcasts, music, audio
  - `lus/` — books, articles
  - `vus/` — films, video

`projects/` — one subfolder per project
  - Contains: SPEC.md, SYSTEM_PROMPT.md, DECISIONS.md, scaffold/

`knowledge/` — cross-project knowledge base
  - Concepts, patterns, and insights that apply across builds

`qui-je-suis/` — Mitch's persistent identity layer
  - `profil-core.md` — identity, citizenship, languages, values
  - `preferences.md` — sleep, food, tools, thinking style
  - `historique-decisions.md` — key decisions and active projects
  - `llm-skills.md` — running log of LLM concepts mastered

`sessions/` — raw infodumps from past sessions
  - Named: YYYY-MM-DD-[topic].md

`taste/` — Taste Vault
  - `entries.md` — one entry per title consumed
  - `manifest.yaml` — relational graph of nodes and edges
  - `session_log.md` — processing session log

## Archive
